1. Wat is het verschil tussen het internet en het world wide web?

  • Het internet is een netwerk wat uit vele kleine netwerken bestaat. Een wereldwijd netwerk bestaat uit miljoenen computers die met elkaar verbonden zijn.
  • Het WWW is een grote verzameling van pagina's die op te vragen zijn.
  • Het internet is de hardware en het WWW is de software.

2. Wat is het verschil tussen HTML, CSS en JavaScript?

  • HTML zijn de bouwstenen van de website, welke informatie te zien moet zijn.
  • CSS is de opmaak van de pagina. Hoe iets eruit moet zien.
  • Javascript maakt de pagina dynamisch, maakt een website interactief

1. Wat is het verschil tussen het internet en het world wide web?

  • Het internet is een netwerk wat uit vele kleine netwerken bestaat. Een wereldwijd netwerk bestaat uit miljoenen computers die met elkaar verbonden zijn.
  • Het WWW is een grote verzameling van pagina's die op te vragen zijn.
  • Het internet is de hardware en het WWW is de software.

2. Wat is het verschil tussen HTML, CSS en JavaScript?

  • HTML zijn de bouwstenen van de website, welke informatie te zien moet zijn.
  • CSS is de opmaak van de pagina. Hoe iets eruit moet zien.
  • Javascript maakt de pagina dynamisch, maakt een website interactief

3. Wat is een HTML tag en een HTML element? Hoe open je een HTML tag en hoe sluit je een HTML tag. Maak hiervoor voorbeelden en leg uit.

Een tag is een opdracht die door een browser kan worden geïnterpreteerd. Een element is een onderdeel van een webpagina. een begintag bepaalt wanneer het onderdeel begint en een eindtag bepaalt wanaar het onderdeel eindigt.
Een begintag is: < >. en een eindtag is < / > Voorbeeld: <h1> Dit is een Header.</h1> Hiermee kan je een titel van een pagina weergeven. Voor de titel van deze pagina is de volgende code gebruikt: <h1>Opdrachten</h1>

4. HTML gebruikt basis tags om structuur op te bouwen. De <html>,<head> en <body>. Geef voor elke element aan waarvoor het dient.

  • <html>: dit element is een container voor alle andere html elementen, dit is de root van een html document.
  • <head>: Dit gedeelte bevat informatie wat niet getoond wordt op de pagina van de web browser, hierbij kan je denken aan: De titel van de pagina, een link naar de CSS, informatie over de auteur en keywords die het document beschrijven.
  • <body>: dit element bevat alle inhoud ven een HTML-document.

5. Op HTML elementen kun je attributen gebruiken. Wat zijn HTML attributen ? Geef minimaal 5 attributen en maak voorbeelden waar je de attributen in toepast.

Attributen worden gespecificeert in de begin-tag. Een attribuut geeft extra informatie over een element.

  • style: Hiermee worden elementen gestyled. <p style="color:red;">Voorbeeld</p>Dit geeft: Voorbeeld
  • alt: Dit wordt gebruikt als alternatieve tekst bij een afbeelding wanneer de afbeelding niet getoond kan worden.

  • class: dit geeft de mogelijkheid om bepaalde eigenschappen aan een element te koppelen. <p class="achtergrond"> Een stukje tekst.</p>. waarbij de class"achtergrond" gestyled kan worden met CSS
  • href: Wordt gebruikt om te verwijzen naar iets. <a href="https://www.buienrader.nl/">buienradar</a> linkt naar buienradar
  • source/ src: Dit is de plaats van een afbeelding, video of audio. zie voorbeeld bij de alt attribute

6. Waarom is een goede HTML Structuur belangrijk? Is deze code een goede structuur? Leg uit.

Een goede structuur is belangrijk zodat bezoekers alle nodige informatie snel kunnen vinden. De code waarnaar gelinkt wordt heeft geen goede structuur. Er missen titels, paragraven, lijsten waardoor het onduidelijk wordt.

7. Wat is het verschil tussen inline en block elementen?

Een block-level element begint altijd op een nieuwe regel en neemt de volle breedte van het browservenster in. Het strekt zich uit naar links en rechts zo ver als het kan. Een inline element start niet op een nieuwe regel en beslaat slechts zo veel breedte als het nodig heeft om wat erin staat weer te geven.

8. Wat betekent nesting? Codeer een voorbeeld met nesting erin.

Bij nesting zit er één of meerdere elementen in een ander element. In onderstaand voorbeeld zitten de heading en paragraph nested in de body.
<body>
<h1>Heading</h1>
<p>Paragraph</p>
</body>

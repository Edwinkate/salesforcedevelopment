Opdrachten

1. Bij het inladen van een bestand is het noodzakelijk om de locatie van het bestand aan te geven. Dit kan op 2 manieren: Wat houden deze 2 paths in?

  • Absolute path: Een absolute path is een volledige URL naar een extern bestand.
  • Relative path: Een relative path verwijst naar een bestand van de huidige pagina.

2. Zoek uit welke formaat afbeeldingen er nog meer kan zijn.

APNG, GIF, ICO, JPEG, PNG, SVG.

3. Zoek uit welke browsers er zijn en maak een lijst van je antwoord.

  • Chrome
  • Safari
  • Firefox
  • Edge
  • Internet Explorer
  • Opera

4. Onderzoek hoe het plaatsen van een foto gedaan wordt en plaats deze boven en onderaan je website door zowel gebruik te maken van de absolute path en relative path.
